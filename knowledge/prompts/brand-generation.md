# Brand Generation Prompt Contract

For each page, generate:

- title
- meta description
- canonical path
- H1
- H2/H3 outline
- body copy
- tables/lists
- CTA copy
- internal links
- unresolved fact notes
- schema recommendation

The model must load only relevant page rules and facts before writing.

## Page Order

1. homepage
2. bonus
3. app
4. payments
5. registration
6. service pages

After each page, run the brand quality gate before continuing.
