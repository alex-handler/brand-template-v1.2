# Anti-Footprint Quality Rules

Brand launches must vary:

- hero composition
- section order within allowed page contracts
- table captions
- FAQ wording
- CTA support copy
- media placement
- palette
- heading phrasing

Do not vary:

- factual source boundaries
- compliance warnings
- canonical URLs
- schema IDs
- affiliate redirect paths

Audit for repeated n-grams, duplicated titles, duplicated meta descriptions, repeated image alt/title text, and identical page-section order across generated sites.
